Sistema de gestão de finanças

Esse foi um projeto desenvolvido em react, typescript e java. Se trata de um sistema para gerenciamento de clientes e o ciclo de vida de suas respectivas parcelas após a compra de algum serviço. Ao decorrer desse projeto atuei no front-end na parte de integração, lógica de alguns componentes, autenticação e autorização, e no back-end na parte de lógica das regras de negócio, banco de dados, autorização e autenticação, também exerci os papéis de master e PO do projeto (em diferentes sprints) fazendo reuniões com o time, criando algumas documentações, conversando com o cliente, montando backlogs e diagramas BPMN.

Classroom - organização escolar

Classroom é um projeto desenvolvido em react e typescript. É uma plataforma que visa disponibilizar ferramentas para consultas para os diferentes públicos envolvidos num ambiente escolar. Para alunos será possível visualizar notas, horários de aulas e etc, para a diretoria será possível fazer operações de matrícula, professores poderão organizar matérias e administradores farão operações envolvendo os usuários em geral. A ideia com essa aplicação é aumentar a proficiência com as tecnologias envolvidas e fazer o desenvolvimento do sistema baseando-se em boas práticas, como uma arquitetura pautada em testes e um foco maior na segurança da aplicação. Esse projeto abrange todo escopo de uma REST API abordando também etapas como o design das páginas, organização de projeto e, principalmente, documentação detalhada.
